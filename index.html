Git -版本控制工具

分支

  • 查看分支：git branch，当前分支会标有一个*

  • 创建分支：git branch [分支名称]

    • 分支中的代码，与创建那一刻主分支中的内容完全相同

  • 切换分支：git checkout [分支名称]

  • (简写)创建并切换分支：git checkout -b [分支名称]

  • 合并分支：git merge [分支名称]，即：将其他分支合并到当前分支

  • 删除分支：git branch -d [分支名称]
